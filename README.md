# Ordeal
Makes integers into ordinal strings.
For example, 81 -> "eighty-first". 
But does not make ordinal numbers (ex. 81 -> 81st)

The package can be installed by adding `ordeal` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ordeal, "~> 0.1.0"}
  ]
end
```
